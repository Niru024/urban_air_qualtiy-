Urban Air Quality Monitoring System

The Urban Air Quality Monitoring System is a web-based application that provides real-time Air Quality Index (AQI) monitoring and pollutant analysis for urban areas. The system displays pollutant data such as CO, NO₂, PM2.5, PM10, and O₃ using interactive dashboards, charts, and maps to help users understand pollution levels and take necessary health precautions.

Features
Real-time AQI monitoring
Pollutant-wise analysis
Interactive charts and dashboards
Location-based AQI visualization
User-friendly responsive UI
AQI prediction using basic Machine Learning concepts
Smooth animations and modern UI design using Spline
